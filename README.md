Roxiler System Task - Frontend



  Tech Stack

-  React 19
-  Tailwind CSS
-  Vite
-  Redux Toolkit
-  React Router v7
-  Chart.js
-  Lucide Icons & Hero Icons
-  React Hook Form
-  react-toastify

Installation

Clone the repository
  
   git clone <https://github.com/Hanumant-Pisal/rx_system_task_frontend>
  

Install dependencies
   
   npm install
  

 environment variables
  
   VITE_API_BASE_URL=
   

Start the development server
  
   npm run dev
  

Build for production
  
   npm run build
 

    

 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
